Welcome to AnkaaCMS Community Edition.

AnkaaCMS is a multilingual CMS which is currently in development and alpha stage. You should not use this CMS in a production environment, however we would like you to join the development.

If you require help, contact @mrdemc or go to https://dienstkoning.nl/ and leave us a message using the contact-page.
